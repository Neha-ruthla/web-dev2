Project Overview :
Smart Event Dashboard is a simple web application to manage events.
Users can add events with title, date, category, and description, view them in a list, clear all events, and also add sample events for demo purposes.
It also demonstrates basic DOM manipulation in JavaScript.


Features:
Add new events with details
Display events dynamically
Clear all events option
Add sample events quickly
DOM manipulation demo (innerHTML, innerText, textContent)
Key press detection demo

Technologies Used :
HTML
CSS
JavaScript

How to Run :
Download or clone this repository
Open the project folder
Run index.html in your browser
